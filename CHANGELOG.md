### Added
- Initial release of TGBot4SillyTavern
- Character selection via Telegram inline keyboard
- Chat history management with pagination support
- Message editing functionality
- Message regeneration functionality
- Completion preset selection
- HTML to Telegram format converter for rich text support
- Docker deployment support with docker-compose
- Comprehensive documentation (README, USAGE, CONTRIBUTING)
- GitHub Actions CI/CD workflows
- Makefile for development tasks
- Interactive setup script
- Unit tests for HTML formatter

### Features
- Headless Chromium browser automation for SillyTavern
- Telegram Bot API integration
- Button-based user interface
- Support for bold, italic, and link formatting
- Graceful error handling
- Configurable via environment variables
